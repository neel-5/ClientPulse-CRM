# Video Guide

Record at 1080p with the browser at 100% zoom. Hide bookmarks and personal notifications.

## Capture order

1. Login with the demo account.
2. Show the overview metrics and explain the business problem.
3. Create a lead from a mock WhatsApp message.
4. Open the inbox and generate an AI reply draft.
5. Move the lead to `Interested` in the pipeline.
6. Schedule a follow-up and show it in the agenda.
7. Run Sheets sync and open the workbook template.
8. Show automation logs and one Python script.
9. Show the public GitHub repository, tests, and architecture.
10. Show honest pilot evidence using `product/REAL_USER_VALIDATION.md`.

Use `product/VIDEO_DEMO_SCRIPT.md` for the two-minute narration. Avoid claiming that mock
integrations or demo users are live customers.
